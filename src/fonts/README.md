# Font Installation Instructions

To use the Jameel Noori Nastaleeq font in this project:

1. Download the Jameel Noori Nastaleeq TrueType font file (.ttf)
2. Place the font file in this directory
3. Rename the file to `Jameel-Noori-Nastaleeq.ttf`

## How to obtain the font:
- Visit a trusted Urdu font repository
- Search for "Jameel Noori Nastaleeq" or "Jameel Noori Nastaleeq.ttf"
- Download the font file ensuring it's licensed appropriately for your use
- Add it to this directory

## Note:
The application is configured to use this font file in `src/app/layout.tsx`. Once you add the font file, the application will automatically use it for Urdu text.
